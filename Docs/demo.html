MoogressBar is a small javascript extension for mootools.

How to use

At first you have to include the moogressBar.js file after your mootools framework files: <script type="text/javascript" src="js/mootools-core.js"></script>
<script type="text/javascript" src="js/moogressBar.js"></script> After that you can easily init the moogressBar by using the following code: var myMoogressBar = new MoogressBar(); Set a percent value with the setPercentage() function: myMoogressBar.setPercentage(20); Check the moogressBar.txt for more documentation.
